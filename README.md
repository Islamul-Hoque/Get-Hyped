# Get Hyped - Social-first Content Agency

A modern, responsive web project for **Get Hyped**, a social-first content agency.  
This project delivers animated, strategy-driven content sections with a polished UI/UX using **Next.js, TypeScript, Framer Motion, GSAP, Lenis, and more**.

## ✨ Key Features

- **Hero Section** — GSAP‑animated interactive text/video cards with random rotation effects in a responsive grid, plus a bottom statement block showcasing the agency’s content philosophy.

- **Intro Section** — Responsive desktop‑only intro with image, strategic content text, and framer‑motion animated CTA buttons featuring subtle rotation and arrow transitions.

- **MobileIntro Section** — Mobile‑only intro with motion‑animated video, descriptive text, and framer‑motion CTA button featuring subtle rotation and arrow transitions.

- **ExpertiseStack Section** — Scroll‑responsive expertise cards with motion‑animated scaling and alternating rotation, combining video previews, descriptive text, and CTA buttons linking to detailed expertise pages.

- **ContentScoring Section** — Dynamic showcase of scoring cards with scroll‑responsive positioning, hover‑activated video playback, immersive 3D tilt effects, and branded CTA buttons highlighting impactful client success stories.

- **BrandSlider Section** — Auto‑scrolling carousel of partner brand logos built with Embla Carousel and Framer Motion, featuring subtle rotation effects on drag interaction for an engaging showcase.

- **Footer Section** — Interactive hero with hover‑triggered floating logo animations, scroll‑based rotating footer badge, animated CTA button, and responsive layout combining logo and right‑side content.

## 🛠️ Tech Stack

- **Framework**: Next.js  
- **Language**: TypeScript  
- **Animations**: Framer Motion, GSAP, React Spring  
- **Scrolling**: Lenis (smooth scrolling)  
- **Carousel**: Embla Carousel React + Auto Scroll plugin 
* **Icons:** Lucide React, React Icons
* **Deployment:** vercel

## ⚙️ Installation & Local Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Islamul-Hoque/Get-Hyped.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd get-hyped
    ```

3.  **Install dependencies:** 
    ```bash
    npm install
    ``` 

4.  **Start the development server:**
    ```bash
    npm run dev
    ``` 

5.  **Build for production:**
    ```bash
    npm run build
    ```
